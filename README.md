# node_craigslist_scapper
using node.js to scrape craigslist. 
Decided to mess with some node.js and created a web scrapper to scrape the tech jobs from craigslist.
Scrapped: 
    the title, 
    date posted,
    description,
    location(area),
    salary, 
    address,
    url.
